# LingCalc - Your Online Text Calculator

## 简介

**LingCalc** 是一个简洁而强大的在线文本计算器，灵感来源于 [Numi.app](https://www.numi.app/)，旨在为用户提供一个直观且易于使用的方式来执行数学运算。它支持广泛的数学表达式，从基础的算术到变量函数，旨在满足各种计算需求。

演示地址： https://cal.zengqueling.com 

## 特点

- **易于使用**：用户友好的界面，无需任何数学软件使用经验。
- **即时反馈**：输入表达式后，结果即刻呈现。复制后结果带完整等式。
- **功能强大**：支持广泛的数学运算，包括但不限于基础运算、使用变量等。(单位转换，汇率转换debuging..)
- **完全开源**：项目代码完全开放，欢迎社区贡献和改进。

## 使用

只需在计算器中输入您的数学表达式，例如：
```
简单算术
-----------------
一年有多少周：365 / 7 
一个鸡蛋1.5元 * 6个鸡蛋 

使用变量
-------------------
Egg = 1.5 
Carton = 6 
Carton * Egg 
```
计算器将自动计算并显示结果。
